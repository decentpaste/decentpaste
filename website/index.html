Why DecentPaste?

Built for privacy-conscious users who want seamless clipboard sync without compromising security.

Cross-Platform

Works on Windows, macOS, Linux, Android, and iOS. True universal clipboard for all your devices.

Decentralized

No cloud servers, no accounts, no subscriptions. Direct peer-to-peer connection over your local network.

End-to-End Encrypted

Military-grade AES-256-GCM encryption. Your clipboard data is encrypted before it ever leaves your device.

Completely Private

Your data never leaves your local network. No analytics, no telemetry, no data collection. Ever.

Secure Pairing

6-digit PIN verification with X25519 key exchange. Cryptographically secure pairing with no shared secrets transmitted.

Open Source

Fully auditable code on GitHub. Trust, but verify. Built with transparency and community in mind.

Security You Can Trust

DecentPaste is designed with security and privacy at its core. Your clipboard data never touches any cloud server - it stays on your local network.

AES-256-GCM Encryption

Military-grade authenticated encryption protects every clipboard sync between your devices.

X25519 ECDH Key Exchange

Modern elliptic curve cryptography for secure key derivation. Shared secrets are never transmitted over the network.

Local Network Only

Peer-to-peer communication via mDNS discovery. Your data never leaves your local network - no internet required.

100% Open Source

Every line of code is publicly auditable on GitHub. Trust, but verify. No hidden backdoors or data collection.
